git remote -v : git 연결정보 확인
git remote remove 지울거 : git 연결 정보 삭제

1. git repository를 받음(git clone 저장 주소)
2. git remote add origin 저장 주소
3. 받은 폴더에 들어감
4. git status (main으로 떠야됨)
5. git branch GUGU
6. git push origin GUGU
7. git checkout GUGU
8. git add asdf.py
9. git commit -m 메시지
10. git push origin GUGU
11. git rm asdf.py
12. git commit -m Delete
13. git push origin GUGU
